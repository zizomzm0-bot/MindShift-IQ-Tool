# 🧠 Mind Shift | Professional IQ Assessment Tool

Welcome to the official repository of the **Mind Shift** IQ Tool. This is a high-end, bilingual web-based application designed to provide a seamless and elegant user experience for IQ testing.

## ✨ Features
* **Bilingual Support:** Fully functional in both **Arabic** and **English**.
* **Premium UI/UX:** Sophisticated dark theme with a gold aesthetic.
* **Responsive Design:** Optimized for Mobile, Tablet, and Desktop screens.
* **Instant Results:** Real-time scoring system using custom JavaScript logic.
* **Easy Integration:** Lightweight HTML/CSS/JS structure for quick deployment.

## 🛠️ Installation & Usage
1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/](https://github.com/)[Your-Username]/MindShift-IQ-Tool.git
    ```
2.  **Open the project:**
    Simply open `index.html` in any modern web browser.
3.  **Deployment:**
    Upload the files to your server or use platforms like GitHub Pages for hosting.

## 📂 Project Structure
* `/assets`: Contains images and brand logos (Mind Shift).
* `/css`: Custom styling for the gold & black theme.
* `/js`: Core logic for the IQ assessment questions and scoring.
* `index.html`: The main entry point.
